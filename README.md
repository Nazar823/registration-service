# Чтобы запустить проект на локальной машине #
Требуется создать .env файл в корне проекта
В нем указать:

* MY_PORT = your_port
* SECRET_KEY = your_key

Подключение к БД
* DB_USER = your_user
* DB_NAME = your_db_name
* DB_PASSWORD = your_db_password
* DIALECT = postgres //or other
* HOST = your_host
